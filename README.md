# BigSur 11 20B50 OpenCore iMac20,1
Can be used for Catalina

![alt text](https://i.imgur.com/MvtzUNy.jpg)

# Hardware
- Intel i7-10700k
- Asrock z490m Pro4m
- Audio: Realtek ALC1200
- Ethernet: Intel I219-V
- Ram: 32GB
- GPU: RX 5600 XT
- BCM94360CS2 + Adapter M.2 Key E
- SSD 970 Evo Plus

# Working
- Sleep/Wake
- Shutdown
- Restart
- Audio/Mic
- Audio HDMI
- Bluetooth
- Filevault
- Wifi
- AirDrop
- USB map
- GPU RX 5600 XT (Use kext or ssdt)
- Messages/FaceTime
- Fake Retina
